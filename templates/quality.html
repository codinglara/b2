Quality: About This Data Set

A brief summary about the quality of the data set here

The data was generally of good quality, with very few missing values (missing values were possibly created, when the sensors had been turned-off because of battery optimization).
I used wrangler for cleaning up the missing values. As suggested by wrangler, rows with null values were simply deleted.
The timestamp was in milliseconds, and was converted to the corresponding date-time format, using an appropriate 'Calculate Field' command in Tableau.

Is the data complete?

Do the missing values raise any concerns?

The missing values in the initial data do not raise any concerns, because we have very frequent sampling of data, and have many location points wthin the very small time periods

Also, the dataset has been obtained by collecting sensor data from a large number of days. Therefore, even if a few rows are missing, it doesn't hurt the analayisis of the trends in movement.

When I uploaded the dataset on Wrangler, a few empty cells (null values showed up). Since there was no appropriate fill suggestions, the rows with null values were removed.

Is the data coherent?

Does it make sense relative to itself? Does it match your expectations? Are the distributions sensible? What is your evidence for this?

The distributions are sensible, from analysis. I have verified the location stats and they all in and on the route from CMU to my apartment. There are not outlier location instances.

Is the data accountable?

Yes, the data is accountable, because the data was directly sourced from the sensors and no modifications were done prior to data-cleaning.

I have also provided web-links to the sources that were obtained by the probing sensors.

Is the data correct?

What aspects of this data are verifiable? How did you do so? What did you find?

We can verify that the data is accurate, by checking the location plotting on the map in Tableau.

During the exploratory analysis, I sampled the latitudes and longitudes and ensured that they lie reasonably close to the latitude and longitudes of my visited locations during the time that the data for the study was being collected.

Another, possible way of verifying the data would be to measure the altitude values of the visted locations, and verify them with the available geographic data about the locations.
