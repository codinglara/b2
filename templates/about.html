Accountability: About This Data Set

Write a brief summary about the data set here

The dataset lists the stats of all the football players playing in the 2018 Fifa Men's Football World Cup.
It gives us information about the the number of played games, played minutes and the performance attributes of the players

Where did this data come from?

The data was obtained from the website data.world.com.
Link

Who has access to it?

The data is accessible by any member of data.world. In fact, the data has been created by a team at data.world, and is open to be downloaded by any member of the data.world community

Read Access: Anyone can download the dataset from the data.world website.

Write Access: Only the creator(s) team of the dataset can modify it or write over it.
