Accountability: About This Data Set

Write a brief summary about the data set here

The dataset has tracked attributes from 'location', 'linear_accelerometer' and 'gyroscope' sensors from the smartphone.
The data generated by the 'location' sensors have the following attributes:

  • id: the data item collection id
  • timestamp: the timestamp in milliseconds that was subsequently converted to date and time
  • device_id: the unique ID of the device which is being used to collect data
  • latitude: the latitude value when the data item was sent
  • longitude: the longitude value when the data item was sent
  • speed: the speed of movement at the time when the data was collected
  • altitude: the altitude when the movement data was collected
  • provider: the provider of the location data eg. wifi/cellular network
  • accuracy: the estimated accuracy radius of the collected data (lesser accuracy value here, the better the correctness)

The data generated by the 'linear accelerometer' sensors have the following attributes:
  • id: the data item collected id
  • timestamp: the timestamp in milliseconds that was subsequently converted to date and time
  • device_id: the unique ID of the device being used to collect data
  • values 0, 1, 2: values of the gyroscope along 3 axis
  • accuracy: the number of axis along which the data is accurate

The data generated by the 'gyroscope' sensors is very similar to the 'linear accelerometer, and has the following attributes:
  • id: the data item collected id
  • timestamp: the timestamp in milliseconds that was subsequently converted to date and time
  • device_id: the unique ID of the device being used to collect data
  • values 0, 1, 2:values of the gyroscope along 3 axis
  • accuracy: the number of axis along which the data is accurate
